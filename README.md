# Nostra – Fashion E-Commerce Website

## 📌 Project Overview

Nostra is a modern fashion e-commerce website designed to showcase clothing collections with an attractive and responsive user interface. The website allows users to browse new arrivals, most wanted products, and promotional offers. It is built using HTML, CSS, and JavaScript with interactive UI features and theme customization.

## 🚀 Features

* Responsive and modern fashion website design
* Image slider for showcasing featured collections
* Product sections for **New Arrivals** and **Most Wanted**
* Interactive hover effects and smooth animations
* Dark and light theme switching
* Newsletter subscription section
* Attractive promotional offer banner
* Mobile-friendly layout

## 🛠️ Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and responsive design
* **JavaScript (Vanilla JS)** – Interactivity and dynamic behavior
* **Font Awesome** – Icons
* **Google Fonts** – Typography
* **Unsplash Images** – High-quality product images
